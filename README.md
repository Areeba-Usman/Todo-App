# Todo-App
"A simple and interactive Todo App built with HTML, CSS, and JavaScript. Users can add, edit, and delete tasks."


## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive design

## How to Use
1. Clone this repository.
2. Open the `index.html` file in any web browser.
3. Start adding your tasks!

## Technologies Used
- HTML
- CSS
- JavaScript
